    • This text contains a <small> small print element, representing side-comments and legal disclaimers.
    • Use <mark> highlighted to indicate text marked or highlighted for reference.
    • Use <abbr> HTML to mark abbreviations, showing a tooltip on hover.
    • Use <cite> The Web Standards Manual to reference the title of a work.
    • Use <q> inline quotation for inline quotations.
    • Use <dfn> defining instance to mark the defining instance of a term.
